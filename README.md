# logo-harvest

I am creating a SaaS platform for investment bankers and consultants that makes pulling and formatting logos of companies easy. Essentially, the user gives a list of company names or domains, then the software pulls their logos, and auto formats them, then you can export them as a Powerpoint file.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/logo-harvest.git
cd logo-harvest
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
